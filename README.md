# NAICS-code

This repository contains the core Matlab implementation related to the manuscript:

**Non perceptible Adversarial Cryptographic Scheme for Privacy preserving Image Protection**

## Current scope

The current release includes the main cryptographic and steganographic modules used in the manuscript, including:

- DFDM related processing
- encryption and decryption
- embedding and extraction
- main supporting Matlab procedures for the core workflow

## Not included in the current release

The following components are not included in the current repository:

- airport localization module
- surrogate patch generation and image inpainting module

These components are described in the manuscript as separate processing stages.

## Purpose

This repository is intended to support understanding and partial reproduction of the core cryptographic and embedding workflow reported in the paper.

## Environment

- Matlab
